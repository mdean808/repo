Spoke

By Unicorn808

Change Activity Indicator Colors

Description

Spoke is a small, lightweight tweak that changes the colors of all activity indicators in iOS. Tested on iOS 12, will probably work on 11 as well. It should work everywhere, though some apps like Apollo have custom activity indicators.
